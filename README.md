# 校园小吃街小程序

## 项目简介

本项目旨在为校园小吃摊主和学生用户提供一个便捷的线上平台。其核心目的是将三轮车摊位信息线上化，方便学生在宿舍即可浏览摊位信息、了解营业状态，从而避免不必要的线下逛街，节省时间和精力。项目致力于提升校园内小吃交易的效率和便利性。

## 主要功能

### 摊主功能

*   **摊位信息管理**：摊主可以方便地创建、编辑和更新自己的摊位名称、描述、图片等详细信息。
*   **营业状态切换**：摊主可以实时切换摊位的营业状态（“出摊”或“收摊”），并同步展示给用户。
*   **实时订单处理**：(根据实际情况补充，例如：接收和管理用户订单，处理支付等)

### 用户功能

*   **浏览摊位信息**：学生用户可以浏览所有已上线摊位的基本信息，包括名称、状态等。
*   **搜索摊位**：用户可以通过关键词搜索特定摊位，快速找到感兴趣的小吃。
*   **查看营业状态**：用户可以实时查看摊位的营业状态，避免前往后发现摊位未营业。

## 技术栈

本项目主要采用微信小程序和腾讯云开发技术栈，实现前后端一体化开发：

*   **前端**：微信小程序
    *   使用微信小程序原生框架进行开发。
    *   页面采用 WXML、WXSS 和 JavaScript 进行构建。

*   **后端**：腾讯云开发
    *   **云函数**：提供后端业务逻辑支持，如用户注册、摊位数据管理、图片上传、轮播图获取等。
    *   **云数据库**：NoSQL 数据库，用于存储摊位信息、用户信息、轮播图数据等结构化数据。
    *   **云存储**：提供文件存储服务，用于存储摊位图片、用户头像等静态资源。
